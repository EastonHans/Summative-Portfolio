01 / About

Building software with attention to detail.

Software developer based in Nairobi, working across frontend and backend. React and TypeScript on the client side, Django and Flask on the server, with enough machine learning and data analysis to make sense of the numbers behind a product.

Fundamentals come first: clean component architecture, typed interfaces, and systems that hold up under real use. STRYD, a sneaker store SPA built in React and deployed on Vercel, is one example. A BS in Computer Science from CUEA, followed by further software engineering training at Moringa School, backs the rest up.

Beyond client work: co-founding a small web development studio focused on the Kenyan market, and writing up what gets learned along the way.
